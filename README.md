# Wharflab Tap

## How do I install these formulae?

`brew install wharflab/tap/<formula>`

Or `brew tap wharflab/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "wharflab/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
